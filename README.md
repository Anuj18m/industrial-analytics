# Industrial Analytics Platform

This project is an industry-style decision-support system for manufacturing data.

## Features
- Automated data ingestion (no manual entry)
- KPI dashboard with charts
- Rule-based automated insights
- Separate data Q&A chatbot
- Streamlit-based UI

## Tech Stack
- Python
- Pandas
- Streamlit

## How to Run
pip install streamlit pandas
streamlit run app.py
